Программно-вычислительный комплекс для вычисления температурных изменений в системе металл - жидкость.

=====

Автор: Поляков Андрей Владимирович

=====

Тип ЭВМ: IBM PC-совмест. ПК, ARM v6/v7
Язык: Java
ОС: OC Windows XP/7, Linux, MacOS X, Solaris

=====

Программа предназначена для вычисления разности температур металла и охлаждающей жидкости в различные моменты времени, позволяющей оценить температуру металла и темпы его охлаждения, а также для вывода этой информации в виде таблицы и графика зависимости разницы температуры от времени. Эталоном служит математическая модель процесса теплообмена между металлом и жидкостью в замкнутой и теплоизолированной системе. Математическая модель основана на уравнении теплового баланса (без учета теплообмена со средой).  Вычисления проводились методом Эйлера и методом Рунге-Кутта 4го порядка. Реализован вывод в отдельные Excel и PDF документы величин разности температур в определённый момент времени, а также сохранение графиков с текущим масштабом и различными параметрами в стандартных форматах (*.png, *.svg).
